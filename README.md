# ✨ Todo Pro App

A modern and responsive **Todo Application** built using **HTML, CSS, and JavaScript**.
This app helps users manage daily tasks with a clean UI and smooth user experience.

---

## 🚀 Features

* ✅ Add new tasks
* ✏️ Edit tasks (double-click)
* ❌ Delete tasks
* ✔️ Mark tasks as completed
* 🔍 Search functionality
* 🎯 Filter tasks (All / Active / Completed)
* 📊 Task counter (completed vs total)
* 🌙 Dark / Light mode toggle
* 💾 Persistent storage using LocalStorage
* 🎨 Modern UI with animations

## 🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript
